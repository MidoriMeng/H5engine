# 说明
大三第二学期“游戏引擎分析”的作业。以基于H5、javascript游戏引擎“白鹭引擎”为参考对象，学习游戏引擎的编写。engine是引擎所在文件夹，test和Resources分别是测试用例和用例所需的资源文件。
# HOWTO
下载解压后在终端输入“tsc -p engine”和“tsc -p test”，运行index.html
